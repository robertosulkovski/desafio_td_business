# Desafio Técnico - Analista de Dados

## Objetivo

Construir uma solução analítica utilizando dados da RAIS 2020, 2021 e 2022 para apoiar análises do mercado de trabalho.

## Tecnologias

- Python
- Pandas
- Jupyter Notebook
- Power BI
- Git/GitHub

## Etapas

- Inventário e compreensão dos dados
- Tratamento e padronização
- Consolidação das bases
- Modelagem dimensional
- Construção do modelo estrela
- Desenvolvimento dos dashboards
